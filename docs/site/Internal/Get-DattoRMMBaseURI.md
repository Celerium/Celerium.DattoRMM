# Get-DattoRMMBaseURI

## SYNOPSIS
Shows the DattoRMM base URI

## SYNTAX

```powershell
Get-DattoRMMBaseURI [-AndApiUri] [<CommonParameters>]
```

## DESCRIPTION
The Get-DattoRMMBaseURI cmdlet shows the DattoRMM base URI from
the global variable

## EXAMPLES

### EXAMPLE 1
```powershell
Get-DattoRMMBaseURI
```

Shows the DattoRMM base URI value defined in the global variable
